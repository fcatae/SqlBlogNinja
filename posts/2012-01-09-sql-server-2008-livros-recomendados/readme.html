SQL Server 2008 - Livros Recomendados

Consultando a lista de livros recomendados para o Microsoft Certified Master (MCM), encontrei as seguintes recomendações:

 

  • Microsoft SQL Server 2008 Internals

  • Inside Microsoft SQL Server 2008: T-SQL Querying

  • Inside Microsoft SQL Server 2008: T-SQL Programming (barnesandnoble.com)

  • Professional SQL Server 2008 Internals and Troubleshooting (barnesandnoble.com)

  • SQL Server 2005 Practical Troubleshooting: The Database Engine (barnesandnoble.com)

  • SQL Server 2008 Query Performance Tuning Distilled (barnesandnoble.com)

  • Pro SQL Server 2008 Failover Clustering (barnesandnoble.com)

  • Pro SQL Server 2008 Mirroring (barnesandnoble.com)

  • Expert SQL Server 2005 Development (barnesandnoble.com)

  • SQL Server MVP Deep Dives (barnesandnoble.com)

  • Microsoft SQL Server 2008 Management and Administration (barnesandnoble.com)

  • Pro SQL Server 2008 XML (barnesandnoble.com)

  • Pro SQL Server 2008 Service Broker (barnesandnoble.com)

  • Pro SQL Server 2005 Assemblies (barnesandnoble.com)

  • Defensive Database Programming with SQL Server (barnesandnoble.com)

  • SQL Antipatterns: Avoiding the Pitfalls of Database Programming (barnesandnoble.com)

 

É praticamente impossível ler todos esses livros . Se você tiver interesse em prestar para a prova de certificação do MCM, recomendaria prestar bastante atenção nos 4 primeiros. Eles são livros que cobrem o fundamento do SQL Server sem aprofundar em uma feature específica. Isso não invalida a leitura dos demais… mas haja tempo!
