TechEd Brasil 2011

Estes dias estava retomando os projetos antigos e um deles era o Blog ninja. Resolvi aliviar um pouco o trabalho e voltar a investir o tempo em escrever mais sobre SQL Server e seu funcionamento interno. Acabei lembrando do TechEd 2011, na qual apresentei em conjunto com o Luiz Felipe Pimenta a palestra “Raio-X do SQL Server: Arquitetura Interna do gerenciador de Banco de Dados”. Que desafio! A idéia era apresentar um conteúdo nível 400 como se fosse 200. Rapidamente passamos sobre os tópicos:

  • Armazenamento de dados (Pages, Extents)
  • Tipos de páginas (IAM, GAM, SGAM, PFS, dados)
  • Índices clustered e não-clustered
  • Buffer Manager
    • Lazy Writer
    • Checkpoint
  • Transaction Log
  • SQL OS
    • Memory Clerks
    • Memory Brokers
    • Scheduler
    • Worker Thread
  • Compilador
    • Parser Tree
    • Compiled Plan
    • Paralelismo
  • Estatísticas
